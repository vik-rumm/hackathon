# AI Money Coach — Hackathon-ready Dashboard

Modern AI-powered personal finance dashboard (dark futuristic UI) with:
- Financial Health Score (0–100) circular indicator
- AI-generated summary card
- Income / Expense / Investment stat cards
- Expense breakdown pie chart + legend with percentages
- Monthly budget tracker with 80% alert + remaining amount
- Subscription tracker list + total cost
- Investments vs Expenses bar chart
- Upload PDF/Image with “AI analyzing document…” status simulation
- AI chat assistant with clean chat bubbles + quick prompts
- Sidebar navigation (Dashboard / Transactions / Subscriptions / Investments / Settings)

## Run (no install)

Open `index.html` in your browser.

Recommended (for best local experience):
- VS Code → install **Live Server** extension → right click `index.html` → “Open with Live Server”

## Files

- `index.html`: layout + sections
- `styles.css`: theme, cards, animations, responsive layout
- `app.js`: demo finance data, charts, upload simulation, chat assistant, routing

## Publish a shareable link (your “project link”)

### Option A — GitHub Pages (free)
1. Create a GitHub repo and upload these 4 files (`index.html`, `styles.css`, `app.js`, `README.md`).
2. In GitHub repo → **Settings** → **Pages**
3. Source: **Deploy from a branch**
4. Branch: `main` / folder: `/root`
5. Save → you’ll get a public link like `https://<username>.github.io/<repo>/`

### Option B — Netlify Drop (fastest)
1. Go to Netlify and open “Deploy manually / Drop”
2. Drag the whole project folder
3. Netlify instantly gives you a public URL
